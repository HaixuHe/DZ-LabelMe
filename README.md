# 人工 + AI 波段标注工具

本地网页程序，面向多波段 GeoTIFF 遥感数据，支持像素级人工标注与随机森林 AI 推理的迭代精修流程。

## 主要功能

- **三波段合成预览**：从文件夹中选择任意 3 个波段按 RGB 顺序合成，自动进行 2%98% 百分位拉伸显示。
- **像素级刷写标注**：支持画笔与橡皮工具，鼠标悬停时显示画笔预览圆圈，可设置笔刷半径和透明度。
- **快捷键操作**：`B` 画笔、`E` 橡皮、数字键 `19` 切换类别、`[/]` 调整笔刷大小、`+/-` 调整缩放、空格 + 左键拖动平移、滚轮以鼠标为中心缩放、中键拖动平移。
- **随机森林 AI 推理**：
  - 可自由选择多个训练特征波段（Ctrl/Cmd 多选）。
  - 支持加入邻域纹理特征（均值 + 标准差，可选 33 至 77 邻域）。
  - 置信度阈值（默认 0.95）+ 2020 邻域均质性双重过滤，低置信度区域标记为无效（99）。
  - 推理时所有波段数值完全相同的像素直接标为无效，不参与分类。
  - 训练时自动排除所有波段相同的样本像素。
  - 训练完成后显示精度指标、OOB 评分及各类别像素分布。
- **双图层独立控制**：AI 推理层与人工标注层完全独立，各自有独立的显示开关和透明度滑块，互不干扰。
- **迭代精修流程**：点击"复制 AI 到可编辑层"后自动重置推理状态（值为 99 的无效像素清空），以修正后的标注重新训练，形成完整的人工 + AI 迭代循环。
- **GeoTIFF 导出**：导出当前标注为带地理坐标投影的 GeoTIFF，未标注和无效（99）像素设置为 nodata。

## 目录结构

```
app.py                  # Flask 后端：波段读取、合成、训练推理、GeoTIFF 导出
templates/index.html    # 主页面
static/app.js           # 前端交互逻辑
static/styles.css       # 页面样式
requirements.txt        # Python 依赖
outputs/                # 运行时临时输出目录
```

## 运行方式

```bash
pip install -r requirements.txt
python app.py
```

默认访问地址：`http://127.0.0.1:5000`

## 使用流程

1. 在左侧"自定义文件夹"输入数据路径，点击**扫描**。
2. 在红、绿、蓝下拉框中选择预览波段，点击**加载三波段合成**。
3. 在"标注工具"面板选择类别，使用画笔对图像进行像素级标注（画笔预览圆圈实时显示）。
4. 在"AI 推理"面板选择训练特征波段和纹理邻域大小，点击**训练随机森林并推理**。
5. 推理完成后，通过显示开关和透明度滑块对比人工标注与 AI 结果。
6. 点击**复制 AI 到可编辑层**，将可信结果导入标注层（无效像素自动清空，推理状态重置）。
7. 用画笔或橡皮修正细节，重复步骤 46，持续迭代精化。
8. 完成后点击**导出当前标注**，下载带地理投影的 GeoTIFF 文件。

## 技术说明

| 项目 | 说明 |
|------|------|
| 后端框架 | Flask |
| 分类器 | scikit-learn RandomForestClassifier（`balanced_subsample` 权重，bootstrap + OOB） |
| 地理信息 | rasterio 读写坐标参考，导出时完整保留 CRS 和 Transform |
| 推理进度 | SSE（Server-Sent Events）流式推送 |
| 无效像素 | 值 `99` 表示低置信度/均质性不足/全波段同值，导出时设为 nodata |
| 渲染 | 双 Canvas 独立图层，`image-rendering: pixelated` 保证像素对齐 |
| 每类采样上限 | 默认 15000，防止复制整图 AI 结果后训练样本膨胀 |
| 波段对齐 | 不同尺寸波段自动按第一特征波段尺寸重采样对齐 |
