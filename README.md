# 人工 + AI 波段标注工具

这是一个本地网页程序，面向当前工作区里的多波段 GeoTIFF 数据，提供以下能力：

- 从一个文件夹中选择任意 3 个波段，按红绿蓝顺序生成合成图。
- 对每个波段做 2% 到 98% 百分位拉伸后显示。
- 在浏览器里进行像素级刷写标注。
- 基于当前 3 个原始谱段和手工标注类别训练随机森林，并进行整图推理。
- 将 AI 推理结果复制到可编辑层，继续修正后再次训练，形成人工 + AI 迭代流程。

## 目录结构

- `app.py`: Flask 后端，负责波段读取、合成、训练和推理接口。
- `templates/index.html`: 主页面。
- `static/app.js`: 交互逻辑，包括像素级刷写、AI 结果叠加、导出掩膜。
- `static/styles.css`: 页面样式。

## 运行方式

在 `rf_label_web` 目录下执行：

```bash
python app.py
```

默认启动在：

```text
http://127.0.0.1:5000
```

如果当前环境缺少依赖，可以先安装：

```bash
pip install -r requirements.txt
```

## 使用流程

1. 进入页面后，左侧会自动扫描工作区内包含波段 tif 的场景文件夹。
2. 选择一个场景，或手动输入自定义文件夹路径并点击“扫描”。
3. 在红、绿、蓝三个下拉框里指定要合成的 3 个波段，点击“加载三波段合成”。
4. 使用画笔对不同类别进行像素级标注。
5. 点击“训练随机森林并推理”，等待整图推理完成。
6. 如果 AI 结果大体可用，点击“复制 AI 到可编辑层”，再用画笔或橡皮修正细节。
7. 重复训练，持续细化结果。
8. 可使用“导出当前标注”或“导出 AI 结果”下载灰度 PNG 掩膜。

## 说明

- 当前模型只使用你选中的 3 个波段作为特征。
- 训练时会按类别做随机抽样，避免在你把整张 AI 结果复制回可编辑层后，训练样本量过大。
- 掩膜文件中 `0` 表示未标注，其余灰度值表示类别编号。
- 如果不同波段尺寸不一致，程序会自动按第一选择波段的尺寸做重采样对齐。
